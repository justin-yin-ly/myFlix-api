myFlix API

Welcome! This is an API used to provide and display information about different movies, directors, and genres.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get full list of movies /movies GET None A JSON Object containing data of all movies
Get data about a single movie by title /movies/[Title] GET None A JSON Object containing data of a single movie, with properties for the description, genre, director, image, and whether it's currently featured. Example:
{
"Title" : "",
"Description" : "",
"Genre" : "{}",
"Director" : "",
"ImageURL" : "",
"Featured" : false
}
Get full list of genres /genres GET None A JSON Object containing data of all genres
Get data about a genre by name /genres/[name] GET None A JSON Object containing data about a given genre, with properties for the genre name and a brief description. Example:
{
"Name" : "",
"Description" : ""
}
Get full list of directors /directors GET None A JSON Object containing data of all directors
Get data about a director by name /directors/[name] GET None A JSON Object containing data about a given director, with properties for the director's name, a brief description, and the movies they've directed. Example:
{
"Name" : "",
"Description" : "",
"Movies" : ""
}
Register a new user /users POST A JSON Object holding data about the user to add, structured as such: {
"Username" : "",
"Password" : "",
"Email" : ""
} 		A JSON Object holding data about the user that just registered, including an ID: {
"id" : #,
"Username" : "",
"Password" : "",
"Email" : ""
}
Update a user's info /users/[Username] PUT A JSON Object holding new data to put onto an existing user, structured as such: {
"Username" : "",
"Password" : "",
"Email" : ""
} 		A JSON Object holding data about the user whose info was just updated, including an ID: {
"id" : #,
"Username" : "",
"Password" : "",
"Email" : ""
}
Get full list of users /users GET None A JSON Object containing data of all existing users
Get a specific user by their username /users/[Username] GET None A JSON Object containing data of the specified user
Add a movie to user's list of favorite movies /users/[Username]/favorites/[MovieID] POST None A text message indicating whether the desired movie was successfully added to the given user's favorite list.
Remove a movie to user's list of favorite movies /users/[Username]/favorites/[MovieID] DELETE None A text message indicating whether the desired movie was successfully removed from the given user's favorite list.
Remove an existing user from the resgistry /users/[Username] DELETE None A text message indicating whether the given user was successfully removed from the registry.
